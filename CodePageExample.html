ASP.NET C# Example

Note: ASP.NET pages require a server (IIS) to run. This page shows what the code looks like and what output it would produce on a server.

The Code (.aspx)

<%@ Page Language="C#" %> <!DOCTYPE html> <html> <body> <h1>Hello from ASP.NET with C#</h1> <% // C# code block string name = "Janna"; Response.Write("Hello, " + name + "!<br>"); // Loop example for (int i = 1; i <= 5; i++) { Response.Write("Number: " + i + "<br>"); } %> </body> </html>

The Output (what the server would return)

Hello from ASP.NET with C#

Hello, Janna!
Number: 1
